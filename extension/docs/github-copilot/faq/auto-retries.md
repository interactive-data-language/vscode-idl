# Auto-Retries

By default, GitHub Copilot will automatically retry when errors occur. This is correct, and ideal, behavior and is where LLMs really shine to work through issues on-the-fly.

However, if you don't want LLMs to do this, then you should clearly express this when you ask your question.

Ex: If you encounter an error, stop processing and let me know.

This can be useful if you have an explicit set of steps you want the LLM to follow without any improvisation.
