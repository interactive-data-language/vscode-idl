# Updating IDL for VSCode and GitHub Copilot

## Key Takeaways

Keep your GitHub Copilot tools up to date to access the latest features and improvements. IDL for VSCode automatically updates MCP tools when the extension updates.

**What Happens:** MCP tools refresh automatically with extension updates

**When to Verify:** After updating IDL for VSCode or when new features are announced

## Automatic Tool Updates

When you update IDL for VSCode, the MCP tools available to GitHub Copilot automatically refresh. This ensures GitHub Copilot always has access to the latest capabilities without manual intervention.

## Verifying Tool Availability

You can confirm GitHub Copilot has access to the latest tools by viewing the tool list in agent mode:

1. Open a chat in agent mode (look for the hexagon icon)
2. Click the tools icon to view available MCP tools
3. Look for tools prefixed with `mcp_idl_` or `mcp_envi_`

If you don't see the expected tools after an update, try restarting VSCode to ensure the extension fully reloads.

## Getting New Features

As new IDL and ENVI capabilities are released, the corresponding MCP tools become available automatically. Check the extension changelog to learn about new features and how to use them with GitHub Copilot.
