# Agent Instructions for IDL Extension Development

## MANDATORY Instructions for .pro and .idlnb files

Before responding to ANY question or request involving .pro or .idlnb files:

1. Always read .github/instructions/IDL.instructions.md FIRST
2. Follow all directives in that file including:
   - Reading relevant context files
   - Offering to use official sources before generating code
3. Do not skip this step even for simple questions

This is not optional. This is required for all IDL-related interactions.

## Reference

Full IDL guide available via MCP resources:

- Use the resources-list-all tool to discover available IDL tutorial resources
- Tutorial resources are prefixed with tutorial- (example: tutorial-idl-basics-00-what-is-idl)
- Use resources-get-resource to fetch specific tutorial content when needed
