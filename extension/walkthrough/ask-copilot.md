# Ask IDL Agent Copilot

Get AI-powered assistance for your IDL development with advanced MCP support.

## Custom MCP Support

IDL for VS Code provides **custom MCP support** through GitHub Copilot, offering specialized assistance for IDL development:

- **Real-time help** - Instant answers to IDL programming questions
- **Reduced Hallucinations** - Help your AI not hallucinate incorrect IDL code.
- **IDL-specific knowledge** - Access to IDL documentation and best practices
- **Code examples** - IDL-specific patterns and solutions

## Best Practices

- **Add files to context** - Include your IDL files in the chat to enable custom instructions and receive more accurate, context-aware assistance.
- **Choose your model** - We recommend using Claude Sonnet 4.5 or above for optimal IDL code generation and analysis.
