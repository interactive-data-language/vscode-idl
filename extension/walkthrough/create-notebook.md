# Create an IDL Notebook

Interactive jupiter style notebooks provide a powerful way to develop and document IDL code.

## Getting Started with Notebooks

Click the **Create New IDL Notebook** or name a file with the **.idlnb** filetype to be started.

## Notebook Features

IDL notebooks (`.idlnb` files) support:

- **Interactive cells** - Mix code and markdown for a iterative modern programming approach.
- **Rich output** - Display plots, images, and formatted results in the file itself
- **Inline execution** - Run cells individually or all at once
- **Export options** - Save results as HTML or PDF

## Use For

- Quick and agile data analysis
- Teaching and demonstrations
- Research documentation
- Prototyping algorithms
