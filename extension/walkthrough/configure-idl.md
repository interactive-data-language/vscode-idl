# Configure Settings

Customize IDL for VS Code to match your workflow.

## What You Can Configure

- **Auto-formatting** - Keep your code clean and consistent
- **IDL settings** - Directory, path, and workspace folders
- **IDL console history** - Session history tracking and storage
- **Problem reporting** - Customize code analysis and linting
- **Notebook display** - Cell output and graphics settings
- **Documentation** - Online or local documentation preferences
- **Copilot Support** - GitHub Copilot custom instructions
