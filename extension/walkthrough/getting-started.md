# Welcome to IDL for VS Code!

This extension provides comprehensive support for the Interactive Data Language (IDL) including:

## Core Features

- **Syntax highlighting** - Beautiful, readable code with semantic highlighting
- **AI assistance** - Custom tools for AI supported IDL development
- **Code completion** - IntelliSense for IDL procedures, functions, and variables
- **Debugging capabilities** - Set breakpoints, step through code, inspect variables
- **Integrated IDL terminal** - Run IDL commands directly in VS Code
- **Notebook support** - Interactive `.idlnb` notebooks for data exploration
- **ENVI integration** - Support for ENVI tasks and workflows
