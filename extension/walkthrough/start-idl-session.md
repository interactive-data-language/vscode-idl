# Start an IDL Session

Run IDL commands interactively in the debug console with full debugging capabilities.

## What You Can Do

- **Set breakpoints** - Pause execution and inspect your code
- **Step through code** - Step in, out, and over routines
- **Inspect variables** - View variable values and call stack
- **Compile & run files** - Execute `.pro` files directly
- **Quick reset** - Reset session without restarting IDL
- **Profile code** - Analyze performance with the built-in profiler
