# ENVI Supported Vector Formats

This document lists all vector file formats supported by ENVI for geospatial vector data.

| Format | Read | Write | File Extension |
|--------|------|-------|----------------|
| ENVI vector file (EVF) | • | • | .evf |
| Esri Shapefile (excluding Z and M polygons) | • | • | .shp |
| GeoJSON file | • |  | .json, .geojson |
| OGC GeoPackage vector file | • |  | .gpkg |

**Legend:**
- • = Feature supported
